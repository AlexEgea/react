# 🧠 Random Quote Machine (React)

Aplicación desarrollada con **React** que muestra frases aleatorias y permite compartirlas en Twitter.
Proyecto realizado como parte de la certificación de **Front-End Development Libraries de freeCodeCamp**.

---

## ✨ Características

* 🔁 Generación de frases aleatorias
* 👤 Muestra dinámica de autor
* 🐦 Botón para compartir en Twitter
* 🎨 Diseño moderno y responsive
* ⚡ Renderizado con React
* ✅ Cumple con todos los tests de freeCodeCamp

---

## 🧱 Tecnologías utilizadas

* React
* JavaScript (ES6+)
* HTML5
* CSS3
* Vite

## ⚙️ Instalación y uso

### 1. Clonar el repositorio

```bash
git clone https://github.com/TU-USUARIO/random-quote-machine.git
cd random-quote-machine
```

### 2. Instalar dependencias

```bash
npm install
```

### 3. Ejecutar el proyecto

```bash
npm run dev
```

👉 Abre en el navegador:

```
http://localhost:5173/
```

Este proyecto incluye los tests oficiales de freeCodeCamp:

* Contenedor `#quote-box`
* Texto `#text`
* Autor `#author`
* Botón `#new-quote`
* Enlace `#tweet-quote`

---

## 📌 Notas

* Se utiliza una lista local de frases (puede mejorarse usando una API externa).
* Se ha tenido en cuenta la compatibilidad con los tests de freeCodeCamp.
* Para evitar problemas con React 18, puede usarse `ReactDOM.render`.
