# 🥁 Drum Machine (Batería Electrónica)

Aplicación web interactiva construida con **React** que simula una batería electrónica. Permite reproducir sonidos tanto haciendo clic como usando el teclado.

Proyecto realizado como parte de la certificación de **Front-End Development Libraries** de freeCodeCamp.

---

## 🧠 Funcionalidades

✔️ 9 pads de batería interactivos
✔️ Activación mediante clic o teclado
✔️ Visualización del sonido reproducido
✔️ Diseño responsive
✔️ Compatible con los tests de freeCodeCamp

---

## 🎮 Controles

| Tecla | Sonido     |
| ----- | ---------- |
| Q     | Heater 1   |
| W     | Heater 2   |
| E     | Heater 3   |
| A     | Heater 4   |
| S     | Clap       |
| D     | Open HH    |
| Z     | Kick n Hat |
| X     | Kick       |
| C     | Closed HH  |

También puedes hacer clic directamente en los pads.

## ⚙️ Instalación y uso

1. Clona el repositorio:

```bash
git clone https://github.com/TU-USUARIO/drum-machine.git
```

2. Entra en la carpeta:

```bash
cd drum-machine
```

3. Instala dependencias:

```bash
npm install
```

4. Ejecuta el proyecto:

```bash
npm run dev
```

5. Abre en el navegador:

```
http://localhost:5173
```
