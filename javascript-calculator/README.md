# 🧮 JavaScript Calculator (React)

Proyecto realizado como parte de la certificación de **Front-End Development Libraries** de freeCodeCamp.


## 📌 Descripción

Esta aplicación es una **calculadora funcional** desarrollada con React que cumple todas las historias de usuario requeridas por freeCodeCamp.

Incluye:

* Operaciones básicas: suma, resta, multiplicación y división
* Soporte para números decimales
* Manejo de operadores consecutivos
* Evaluación de expresiones completas (lógica de fórmula)
* Reset completo de la calculadora

---

## 🧠 Funcionalidades clave

✔️ Evita múltiples ceros al inicio
✔️ No permite más de un decimal por número
✔️ Maneja operadores consecutivos correctamente
✔️ Permite continuar operaciones tras pulsar `=`
✔️ Precisión decimal adecuada (hasta 10 decimales)

## ⚙️ Instalación y uso

### 1. Clonar el repositorio

```bash
git clone https://github.com/tu-usuario/javascript-calculator.git
```

### 2. Entrar en el proyecto

```bash
cd javascript-calculator
```

### 3. Instalar dependencias

```bash
npm install
```

### 4. Ejecutar en desarrollo

```bash
npm run dev
```

Abrir en el navegador:

```
http://localhost:5173
```
