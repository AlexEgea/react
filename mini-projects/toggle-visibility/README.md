# 👁️ Toggle Visibility

Aplicación desarrollada con **React** que permite mostrar y ocultar contenido dinámicamente mediante interacción del usuario.

Este proyecto está enfocado en el manejo del estado y el renderizado condicional.

---

## 🚀 Demo

Botón interactivo que alterna la visibilidad de un mensaje en tiempo real.

---

## 🛠️ Tecnologías

* ⚛️ React
* 🟨 JavaScript (ES6+)
* 🌐 HTML5
* 🎨 CSS3

---

## 🧠 Conceptos trabajados

* useState
* Manejo de eventos (`onClick`)
* Renderizado condicional
* Operador ternario
* Componentes funcionales

## ▶️ Cómo ejecutar

1. Clona el repositorio:

```bash
git clone https://github.com/TU-USUARIO/react-mini-projects.git
```

2. Accede al proyecto:

```bash
cd react-mini-projects/toggle-visibility
```

3. Abre `index.html` en el navegador
   (o usa Live Server en VSCode)

## 📌 Descripción

Este proyecto muestra cómo gestionar el estado de un componente en React utilizando `useState` y cómo renderizar contenido de forma condicional en función de ese estado.

Es un ejemplo básico pero esencial para entender la interactividad en aplicaciones React.
