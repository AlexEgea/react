# 🔐 OTP Generator

Aplicación desarrollada con **React** que genera códigos OTP (One-Time Password) de forma dinámica y permite copiarlos al portapapeles.

Este proyecto está enfocado en la manipulación de estado, generación de datos aleatorios y mejora de la experiencia de usuario.

---

## 🚀 Demo

Aplicación interactiva donde el usuario puede:

* Generar códigos OTP aleatorios
* Seleccionar la longitud del código (4, 6 u 8 dígitos)
* Copiar el código al portapapeles con un clic

---

## 🛠️ Tecnologías

* ⚛️ React
* 🟨 JavaScript (ES6+)
* 🌐 HTML5
* 🎨 CSS3

---

## 🧠 Conceptos trabajados

* useState
* Generación de datos aleatorios (`Math.random`)
* Manejo de eventos (`onClick`, `onChange`)
* Select controlado
* Copiado al portapapeles (`navigator.clipboard`)
* Renderizado condicional
* Gestión de estados de UI (feedback al usuario)

## 📌 Descripción

Este proyecto permite generar códigos OTP numéricos de longitud configurable utilizando lógica basada en números aleatorios.

Además, incorpora funcionalidad para copiar el código al portapapeles y proporciona feedback visual al usuario tras la acción.

Es un ejemplo práctico de cómo combinar lógica de negocio con una experiencia de usuario sencilla y efectiva.

---

## 🎯 Objetivo

Practicar la generación dinámica de datos en React y mejorar la interacción del usuario mediante feedback visual y acciones útiles como copiar al portapapeles.
