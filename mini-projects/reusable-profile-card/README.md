# 🧩 Reusable Profile Card

Componente reutilizable desarrollado con **React** para mostrar perfiles de usuario de forma dinámica y escalable.

Este proyecto está enfocado en la creación de componentes reutilizables, el uso de props y el renderizado de listas con buenas prácticas.

---

## 🚀 Demo

Interfaz limpia con tarjetas de usuario generadas dinámicamente a partir de datos.

---

## 🛠️ Tecnologías

* ⚛️ React
* 🟨 JavaScript (ES6+)
* 🌐 HTML5
* 🎨 CSS3

---

## 🧠 Conceptos trabajados

* Componentes funcionales
* Props
* Renderizado dinámico con `.map()`
* Uso de `key` en listas
* Separación de lógica y UI
* Reutilización de componentes

## ▶️ Cómo ejecutar

1. Clona el repositorio:

```bash
git clone https://github.com/TU-USUARIO/react-mini-projects.git
```

2. Accede al proyecto:

```bash
cd react-mini-projects/reusable-profile-card
```

3. Abre `index.html` en el navegador
   (o usa Live Server en VSCode)

## 📌 Descripción

Este proyecto demuestra cómo crear un componente reutilizable (`Card`) capaz de renderizar múltiples perfiles mediante props, manteniendo una estructura limpia y escalable.

Es una base fundamental para aplicaciones más complejas donde se trabaja con listas de datos.
