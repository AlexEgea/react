# 💡 Inspiration Board

Aplicación desarrollada con **React** que permite añadir y eliminar ideas dinámicamente en un tablero interactivo.

Este proyecto está enfocado en la gestión de estado con arrays, manejo de eventos y renderizado dinámico de listas.

---

## 🚀 Demo

Interfaz sencilla e intuitiva donde el usuario puede:

* Añadir nuevas ideas
* Visualizar una lista dinámica
* Eliminar elementos en tiempo real

---

## 🛠️ Tecnologías

* ⚛️ React
* 🟨 JavaScript (ES6+)
* 🌐 HTML5
* 🎨 CSS3

---

## 🧠 Conceptos trabajados

* useState
* Manejo de arrays en estado
* Añadir elementos (`...spread operator`)
* Eliminar elementos (`filter`)
* Inputs controlados
* Eventos (`onClick`, `onChange`)
* Renderizado dinámico con `.map()`

## 📌 Descripción

Este proyecto simula un tablero de ideas donde el usuario puede gestionar contenido de forma dinámica.

Se ha trabajado especialmente el manejo de arrays dentro del estado de React, aplicando buenas prácticas como la inmutabilidad y el uso de métodos como `map` y `filter`.

Es un ejemplo claro de cómo construir funcionalidades tipo CRUD básicas en el frontend.

---

## 🎯 Objetivo

Practicar la manipulación de datos en React y entender cómo actualizar correctamente el estado sin mutaciones directas.
