# ⚛️ React Mini Projects

Colección de proyectos desarrollados con **React** para practicar y consolidar conceptos clave del desarrollo Frontend moderno.

Este repositorio reúne múltiples aplicaciones pequeñas pero completas donde se trabajan **componentes reutilizables, gestión de estado, hooks, eventos, formularios y consumo de APIs**.

---

## 🚀 Proyectos incluidos

### 🔍 Fruit Search App

Aplicación que permite buscar frutas en tiempo real consumiendo una API externa.

* Uso de `useEffect` para peticiones
* Implementación de debounce
* Manejo de estado con `useState`
* Renderizado dinámico de resultados

---

### 🦸 Superhero Application Form

Formulario interactivo para registrar superhéroes.

* Formularios controlados
* Validación de campos
* Uso de checkboxes
* Botón deshabilitado si faltan datos

---

### 💡 Inspiration Board

Tablero donde puedes añadir y eliminar ideas.

* Gestión de arrays en estado
* Renderizado con `.map()`
* Eliminación dinámica de elementos

---

### 🧩 Reusable Profile Card

Componente reutilizable para mostrar perfiles.

* Uso de props
* Renderizado de listas
* Uso correcto de `key`

---

### 🎨 Color Picker

Selector de colores con lógica reutilizable.

* Hook personalizado
* Manejo de estado
* Separación de lógica y UI

---

### 👁️ Toggle Visibility

Componente para mostrar/ocultar contenido.

* Renderizado condicional
* Eventos (`onClick`)
* Uso de operador ternario

---

### 🔐 OTP Generator

Generador de códigos OTP dinámicos.

* Gestión de eventos
* Manipulación de estado
* Lógica de generación

---

### 📌 Reusable Footer

Footer reutilizable para aplicaciones.

* Componentización
* Estructura limpia de UI

---

### 📊 Mega Navbar

Barra de navegación con submenús accesibles.

* Uso de `aria-*`
* Estructura semántica
* Accesibilidad básica

---

## ▶️ Cómo ejecutar los proyectos

1. Clona el repositorio:

```bash
git clone https://github.com/TU-USUARIO/react-mini-projects.git
```

2. Accede a la carpeta del proyecto que quieras ejecutar:

```bash
cd react-mini-projects/nombre-del-proyecto
```

3. Abre el archivo `index.html` en tu navegador
   *(o usa Live Server en VS Code)*
