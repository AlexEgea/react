Colección de pequeños proyectos desarrollados con React para practicar componentes reutilizables, props, estado, eventos, formularios, hooks y consumo de APIs.

## Proyectos incluidos

- Reusable Profile Card
- Inspiration Board
- Toggle Visibility
- Color Picker
- Fruit Search App
- OTP Generator
- Superhero Application Form
- Reusable Footer
- Mega Navbar

## Tecnologías

- React
- JavaScript
- HTML
- CSS
- Fetch API
- Hooks

## Conceptos practicados

- Componentes funcionales
- Props
- useState
- useEffect
- useCallback
- React.memo
- Formularios controlados
- Renderizado condicional
- Listas con map
- Consumo de APIs
