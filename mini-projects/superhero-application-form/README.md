# 🦸 Superhero Application Form

Formulario interactivo desarrollado con **React** que permite registrar un superhéroe mediante diferentes tipos de inputs.

Este proyecto está enfocado en la creación de **formularios controlados, validación y gestión de estado complejo**.

---

## 🚀 Demo

Formulario dinámico donde el usuario puede:

* Introducir nombre de héroe y nombre real
* Seleccionar el origen de poderes
* Elegir múltiples habilidades
* Enviar el formulario solo si todos los campos son válidos

---

## 🛠️ Tecnologías

* ⚛️ React
* 🟨 JavaScript (ES6+)
* 🌐 HTML5
* 🎨 CSS3

---

## 🧠 Conceptos trabajados

* Formularios controlados
* Manejo de inputs (`text`, `select`, `checkbox`)
* useState
* Gestión de arrays en estado
* Validación de formularios
* Botón dinámicamente deshabilitado
* Eventos (`onChange`, `onSubmit`)
* Prevención del comportamiento por defecto (`preventDefault`)

## 📌 Descripción

Este proyecto implementa un formulario completo en React utilizando el patrón de **componentes controlados**, donde cada campo está sincronizado con el estado.

Se incluye validación básica para asegurar que todos los campos estén completos antes de permitir el envío, además de la gestión de múltiples selecciones mediante checkboxes.

Simula un flujo real de aplicaciones frontend donde el usuario interactúa con formularios complejos.

---

## 🎯 Objetivo

Comprender cómo gestionar formularios en React de manera eficiente, manteniendo el control total del estado y aplicando validaciones en tiempo real.
